Шпаргалка с командами Docker

  • docker login - Вход в реестр

    docker login localhost:8080

  • docker pull nginx - Pull (выгрузка из реестра) образа

    docker pull eon01/nginx localhost:5000/myadmin/nginx

  • docker search nginx - Поиск образа

    docker search nginx --filter stars=3 --no-trunc busybox

  • docker push eon01/nginx - Push (загрузка в реестр) образа

    docker push eon01/nginx localhost:5000/myadmin/nginx

  • docker create - Создание контейнера

    docker create -t -i eon01/infinite --name infinite

  • docker run - Первый запуск контейнера

    docker run -it --name infinite -d eon01/infinite

  • docker rename - Переименование контейнера

    docker rename infinite infinity

  • docker rm - Удаление контейнера

    docker rm infinite

  • docker update - Обновление контейнера

    docker update --cpu-shares 512 -m 300M infinite

  • docker start - Запуск остановленного контейнера

    docker start nginx

  • docker stop - Остановка

    docker stop nginx

  • docker restart - Перезагрузка

    docker restart nginx

  • docker wait - Блокировка (до остановки контейнера)

    docker wait nginx

  • docker attach - Подключение к существующему контейнеру

    docker attach nginx

  • docker ps - Работающие контейнеры

    docker ps -a

  • docker logs - Логи контейнера

    docker logs infinite

  • docker inspect infinite - Информация о контейнере

    docker inspect --format '{{ .NetworkSettings.IPAddress }}' $(docker ps -q)

  • docker events - События контейнера

    docker events infinite

  • docker port - Публичные порты

    docker port infinite

  • docker top - Выполняющиеся процессы

    docker top infinite

  • docker stats - Использование ресурсов

    docker stats infinite

  • docker diff - Изменения в файлах или директориях файловой системы контейнера

    docker diff infinite
